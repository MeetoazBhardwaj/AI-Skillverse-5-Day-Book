# Final Project Cards

## Project Card 1: AI Study Kit

Pick one chapter. Create:

- Summary.
- Mind map.
- Flashcards.
- Quiz.
- Mistake checklist.
- Teacher questions.

## Project Card 2: Future Classroom

Create a presentation showing how AI may change learning in the next ten years. Include what should change and what should never change.

## Project Card 3: AI Safety Poster

Design a poster for younger students. Include privacy, honesty, verification, and healthy use.

## Project Card 4: Tool Battle

Compare three AI tools on the same learning task. Rate them on clarity, accuracy, usefulness, safety, and presentation.

## Project Card 5: My Dream AI Helper

Design an imaginary AI tool that would help students learn better. Explain features, safety rules, and how it prevents cheating.

## Project Card 6: Teach a Grandparent

Create a simple guide that explains AI to a grandparent using examples from daily life.
