# Prompt Practice Cards

Cut, copy, or assign these as classroom activities.

## Card 1: The patient tutor

Create a prompt that makes AI teach you a difficult topic slowly and ask questions after each step.

## Card 2: The examiner

Create a prompt that makes AI test you one question at a time and score you strictly.

## Card 3: The diagram maker

Create a prompt that asks AI to explain a topic using a text diagram or Mermaid diagram.

## Card 4: The mistake coach

Create a prompt that asks AI to find the mistake in your answer without directly giving the final answer.

## Card 5: The presentation planner

Create a prompt that turns a topic into a 5-slide school presentation.

## Card 6: The ethics checker

Create a prompt that asks AI whether your planned use is honest and safe.

## Card 7: The revision planner

Create a prompt that builds a 7-day revision plan based on chapters and available time.

## Card 8: The vocabulary builder

Create a prompt that teaches ten new words using meanings, examples, and a quiz.
