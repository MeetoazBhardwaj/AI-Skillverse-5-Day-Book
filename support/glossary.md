# AI Skillverse Glossary

| Term | Simple meaning |
|---|---|
| AI | Technology that can perform tasks that seem to need human-like thinking. |
| Generative AI | AI that can create text, images, code, summaries, or other content. |
| Prompt | The instruction or question given to AI. |
| Prompt engineering | The skill of writing better prompts to get better results. |
| Model | The AI system trained to recognize patterns and respond. |
| Training | The process through which an AI model learns patterns from data. |
| Hallucination | When AI gives an answer that sounds correct but is false or invented. |
| Bias | An unfair or one-sided pattern in an answer or system. |
| Verification | Checking whether information is true using reliable sources. |
| Active recall | Testing yourself from memory instead of only rereading. |
| Flashcard | A question-answer card used for revision. |
| Source | The place where information comes from. |
| Deepfake | Fake audio, image, or video designed to look real. |
| Academic honesty | Doing school work truthfully and giving credit where needed. |
| AI assistant | A tool that can help with writing, explaining, planning, or creating. |
