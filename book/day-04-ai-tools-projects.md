# Day 4: AI Tools for Projects, Presentations, Research, and Creativity

## Opening story: The toolbox, not the carpenter

A carpenter may own the best tools in the world. But tools alone do not build a beautiful table. The carpenter needs judgment, patience, measurement, design, and skill.

AI tools are the same. ChatGPT, Copilot, Gemini, DeepSeek, Gamma, Canva, NotebookLM, Perplexity, and other tools can help you. But they do not replace your judgment.

The student is the creator. AI is the toolbox.


## Tools mentioned in this book

| Tool | Useful for students | Important caution |
|---|---|---|
| ChatGPT | Explanations, quizzes, writing help, brainstorming, coding support, revision plans | Verify facts and do not paste private information. |
| Microsoft Copilot | Web-connected help, writing, Microsoft ecosystem support, browser assistance | Check school policy before using it for assignments. |
| Google Gemini | Brainstorming, summaries, multimodal help, Google ecosystem support | Do not rely on one answer; cross-check. |
| DeepSeek | Reasoning, coding help, explanations, document chat in supported contexts | Be careful with sensitive files. |
| Perplexity | Research-style search with sources | Read the sources, not just the summary. |
| NotebookLM | Learning from uploaded notes or sources | Upload only safe, non-private study material. |
| Gamma | Presentations, visual documents, project reports | Do not let design hide weak understanding. |
| Canva | Posters, infographics, visual summaries | Respect copyright and image usage rights. |
| Napkin AI | Turning ideas into diagrams and visual maps | Check that diagrams actually match the topic. |


## Tool selection guide

| Student need | Best tool type | Example workflow |
|---|---|---|
| Understand a topic | Chat assistant | Ask for simple explanation, examples, quiz. |
| Research with sources | Research assistant | Use Perplexity or search, then read sources. |
| Learn from notes | Source-grounded notebook | Use NotebookLM with safe study material. |
| Make presentation | Deck builder | Use Gamma or Canva, then rewrite in your own words. |
| Make diagrams | Visual thinking tool | Use Napkin AI or Mermaid diagrams. |
| Code help | Chat/coding assistant | Ask for hints and debugging, not full blind copying. |
| Practice speaking | Voice assistant | Ask AI to conduct a viva or interview. |

## How to use ChatGPT for learning

Good uses:

- Explain a concept at your grade level.
- Create practice questions.
- Turn notes into revision tables.
- Role-play as an examiner.
- Give feedback on your answer.
- Help debug code with explanations.

Poor uses:

- Copying full homework.
- Asking for answers without learning.
- Uploading private documents.
- Believing facts without checking.

Prompt:

```text
I want to learn [topic], not copy an answer. Teach me step by step. Ask me questions after each section. If I ask for the direct answer too quickly, remind me to try first.
```

## How to use Microsoft Copilot

Copilot can help with writing, web-connected questions, Microsoft tools, and browser-based assistance. Students can use it for brainstorming, summarizing, drafting outlines, and learning support. If your school uses Microsoft tools, Copilot may fit naturally into that environment.

Student-safe prompt:

```text
Help me create an outline for a school presentation on [topic]. Keep it suitable for Grade [grade]. Include slide titles, key points, and one activity for the audience.
```

## How to use Google Gemini

Gemini can support brainstorming, explaining, planning, and multimodal learning depending on the features available in your region and account. It is useful when students are already using Google apps for school.

Prompt:

```text
Explain [topic] using a simple analogy, then create a table of key terms and meanings. At the end, ask me three questions.
```

## How to use DeepSeek

DeepSeek can be useful for reasoning, explanations, coding support, and structured thinking. It should still be used carefully, especially with private files or school documents.

Prompt:

```text
Help me understand this coding error. First explain what the error means in simple language. Then show me the smallest change needed. Do not rewrite my whole project unless I ask.
```

## How to use Gamma for presentations

Gamma can turn an outline into a polished deck or document. The danger is that beautiful slides can make weak thinking look strong. Always build the idea first.

Presentation workflow:

```text
Topic -> Research -> Outline -> Key points -> Visual plan -> Gamma draft -> Student rewrite -> Teacher check -> Final presentation
```

Prompt for Gamma or any presentation tool:

```text
Create a 7-slide presentation outline on [topic] for Grade [grade]. Slides should include: title, problem, explanation, example, diagram idea, activity, and conclusion. Keep the language simple and accurate.
```

## How to use Perplexity for research

Perplexity-style tools are useful because they often show sources. But a source shown on screen is not automatically a good source. Read the source. Check date, author, and reliability.

Research prompt:

```text
Find beginner-friendly sources on [topic]. Summarize the main points and list what I should verify from my textbook or teacher.
```

## How to use NotebookLM

NotebookLM is useful when you want AI to work from specific notes, PDFs, or study material. It can summarize, quiz, and help you understand uploaded content.

Safety rule:

Do not upload private, confidential, paid, copyrighted, or school-restricted material unless your teacher or parent has allowed it.

Notebook prompt:

```text
Based only on the uploaded notes, make a revision guide with key points, definitions, likely questions, and common mistakes.
```

## How to use Canva and Napkin AI

Canva helps with posters, infographics, and visual design. Napkin AI can help turn text into diagrams. These tools are useful for project work, but the content must still be accurate.

Visual project prompt:

```text
Convert these notes into an infographic plan. Include title, sections, diagram idea, color theme, and short captions. Keep it suitable for school submission.
```

## Project idea: AI-powered chapter rescue

Choose one chapter you find difficult. Build a study kit:

1. One-page summary.
2. Key terms table.
3. Five flashcards.
4. One diagram.
5. Ten quiz questions.
6. Mistake checklist.
7. Three things to ask your teacher.
